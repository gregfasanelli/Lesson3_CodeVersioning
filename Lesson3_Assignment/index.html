Git Basics What is a Version Control System

A version control system that allows others to work on files at the same time without cluttering the files. Git is a main repository that houses all of the project files. Version control solves file sharing problems where team members override eachothers work.

Git Basics What is Git?

Git is a free open source distributed version control system that is established to handle large volumes in a centralized version control center. Companies such as Google and Microsoft use git and it's commands. Git is stored local so every member has their own copy of data, preventing unintentional overrides.

Git in depth Git commit

Git commands such as git commit help you commit to a staging area which is moving and commiting files to the local repository. The main commands are Add, Commit, and Push which helps save and keep files updated within your git group.
